# קלוריקל — Calical

יומן קלוריות חכם בעברית. צלמו את הצלחת או כתבו חופשי מה אכלתם, וה-AI מזהה לבד את המאכלים, הקלוריות ואבות המזון (חלבון / פחמימות / שומן).

אפליקציית עמוד יחיד (single-file), ללא שרת. כל הנתונים — כולל התמונות — נשמרים מקומית בדפדפן (`localStorage`) על המכשיר שלכם בלבד.

## תכונות

- **יומן בסגנון צ'אט** — כותבים "קפוצ'ינו וקרואסון שקדים" או מצרפים תמונה, והיא נוספת ליומן עם הערכים.
- **זיהוי AI** — עם מפתח API (Google Gemini בחינם, או Claude בתשלום לפי שימוש) הזיהוי נעשה אוטומטית מטקסט **או מתמונה**. בלי מפתח — זיהוי טקסט בסיסי ממאגר מובנה של מאכלים ישראליים נפוצים.
- **יעדים ומעקב** — יעד קלוריות ואבות מזון יומי, טבעת התקדמות וגרף שבועי.
- **מצב כהה/בהיר**, RTL מלא, והתקנה למסך הבית באייפון/אנדרואיד.

## הרצה

זהו קובץ `index.html` יחיד — פותחים אותו בדפדפן וזהו. מתארח חינם דרך GitHub Pages.

## הגדרת זיהוי AI

בתוך האפליקציה: ⚙️ הגדרות ← "זיהוי חכם" ← מדביקים מפתח API.
- **Gemini (חינם):** משיגים ב-[aistudio.google.com](https://aistudio.google.com) (מפתח מתחיל ב-`AIza`).
- **Claude:** משיגים ב-[console.anthropic.com](https://console.anthropic.com) (מפתח מתחיל ב-`sk-ant`).

המפתח נשמר על המכשיר בלבד ולא נשלח לשום מקום מלבד ספק ה-AI שבחרתם.
